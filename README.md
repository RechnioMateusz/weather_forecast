# weather_forecast

----- ----- ----- ----- -----

Weather forecast station created using:
* 1x RaspberryPi
* 1x BME280
* 1x TSL2561
* 1x YL83
* 3x Diode
* 1x Tact-switch

This station measures every hour values:
* Temperature
* Atmospheric pressure
* Humidity
* Precipitation
* Exposition

In the future weather station will send all this data to my portfolio site.
